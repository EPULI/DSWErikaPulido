ETIQUETAS BASICAS DE HTML

Las etiquetas de HTML es un lenguaje que esta escrito con codigos los cuales están escritos como codigo pero pueden ser leidos sin necesidad de que ya esten copilados de hecho los textos que encontramos en la pagina web estan marcados con el fin de dar una instruccion al navegador web, con el fin de que defina como mostraer el texto. Las estructuras HTML, cuentan con una estructura basica.

FUNCIÓNAMIENTO DE LAS ETIQUETAS HTML

Muchas de las paginas web que nosotros conocemos actulamente cuantan con este sistemas en donde su formato termina en .html o .htm en donde lo que hace un navegador toamrlo como un archivo y mostrarlo en la web para que puedan verlo.

ELEMENTOS HTML

Los elementos de un html estan divididos en tres importantes grupos que som : etiqueta de aqpertura, etiqueta de contenido y etiqueta de cierre en donde cada una cumple un parte fundamnetal dentro del proceso, ya que por ejemplo la etiqueta de apertura indica en doden empieza la etiqueta, la etiqueta contenido en donde realizamos lo que se va a mostrar y la etiqueta de ciere es en donde le damos fin a la etiqueta en donde le damos cierre a la etiqueta con la diferencia que esta va con un / en el nombre de la etiqueta
