ETIQUETAS BASICAS DE HTML

Las etiquetas de HTML es un lenguaje que esta escrito con codigos los cuales están escritos como codigo pero pueden ser leidos sin necesidad de que ya esten copilados de hecho los textos que encontramos en la pagina web estan marcados con el fin de dar una instruccion al navegador web, con el fin de que defina como mostraer el texto. Las estructuras HTML, cuentan con una estructura basica.

FUNCIONAMIENTO DE LAS ETIQUETAS BASICAS

Muchas de las páginas web que nosotros conocemos actualmente cuentan con estos sistemas en donde su formato termina en .html o .htm, y lo que hace un navegador tomarlo como un archivo y mostrarlo en la web para poderlo verlo.

ELEMENTOS DEL HTML

Los elementos de un html están divididos en tres importantes grupos que son: etiqueta de apertura, etiqueta de contenido y etiqueta de cierre en donde cada una cumple un parte fundamental dentro del proceso, ya que por ejemplo la etiqueta de apertura indica en donde empieza la etiqueta, la etiqueta contenido en donde realizamos lo que se va a mostrar y la etiqueta de cierre es en donde le damos fin a la etiqueta en donde le damos cierre a la etiqueta con la diferencia que esta va con un / en el nombre de la etiqueta
